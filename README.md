# 🎓 Smart Student Course Portal (Angular Project)

## 📌 Overview
This project is a Smart Student Course Portal built using Angular. It allows students to view available courses, enroll in them, track progress, and search/filter courses. The project demonstrates important Angular concepts such as routing, lazy loading, state management, and pipes.

---

## 🚀 Features
- Dashboard with course progress  
- Course listing and enrollment  
- Search and filtering functionality  
- State management using RxJS (BehaviorSubject)  
- Lazy loading for improved performance  
- Routing between modules  

---

## 🛠️ Technologies Used
- Angular  
- TypeScript  
- HTML5  
- CSS3  
- RxJS

---

## 🧠 State Management
State is handled using RxJS BehaviorSubject:
- Data is stored in a service  
- Components subscribe to updates  
- Ensures real-time data flow across the application  

---

## 🔍 Custom Pipe
A custom filter pipe is implemented to search courses dynamically based on user input.
